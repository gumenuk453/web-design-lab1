Про MyContacts

Твій надійний та зручний телефонний довідник для організації контактів

MyContacts — це сучасний веб-додаток, створений для зручного зберігання та управління вашими телефонними контактами.

Забудьте про загублені номери чи незручні паперові записники. Зберігайте всі необхідні дані в одному надійному місці.

Основні можливості:

  • Додавання, вилучення та редагування абонентів (прізвище та список телефонних номерів).
  • Автоматичне групування та сортування контактів за першою літерою прізвища.
  • Швидкий пошук потрібного номера.
  • Доступ до довідника з будь-якого пристрою.

Управління контактами просте та інтуїтивно зрозуміле:

  • Авторизація:
    Зареєструйтесь або увійдіть, щоб створити свою персональну телефонну книгу.
  • Створення контакту:
    Натисніть кнопку додавання, введіть прізвище абонента та додайте один або кілька телефонних номерів.
  • Перегляд та сортування:
    На головній сторінці ви побачите всі ваші контакти, автоматично згруповані за алфавітом для зручної навігації.
  • Редагування:
    У будь-який момент ви можете оновити номер телефону або видалити неактуальний контакт.

Цей проєкт розроблено з використанням сучасного стеку веб-технологій (Frontend), що забезпечує швидку роботу та адаптивність на будь-яких екранах.

Основний стек:
  • HTML5 — семантична структура сторінок.
  • Bootstrap 5.3 — потужний CSS-фреймворк для стилізації та сітки.
Дизайн та UI:
  • Mobile First — повна адаптація під смартфони та планшети.
  • Bootstrap Icons — векторна графіка та іконки для кращого сприйняття.
Проєкт створено в середовищі Visual Studio Code в рамках лабораторної роботи з курсу "Web-дизайн"
